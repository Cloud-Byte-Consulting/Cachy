# Cachy Core Platform Pod

This pod defines the operating behavior for building Cachy.

## Files

- `pod.md` - pod identity, ownership, scope, and write policy.
- `behavior.md` - prioritization, triage, and escalation rules.
- `sources.md` - evidence sources and required fields.
- `workflows.md` - project playbooks and completion criteria.

## Scope

The pod covers the Go CLI/proxy, Electron companion app, WASM plugin system, agent integrations,
cross-platform packaging, security/privacy, and architecture documentation.
